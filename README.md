# GADGETS

自用小工具库.

## ImgRotateTool

- 批量将文件夹中图片旋转 90°.


## NanoVideoManager
### Merge files: 
- 将指定文件夹中的视频整合在一起.

### JellyFin Manager:
- Generate thumbnails: 待调试. 计划删除.
- Deleting nfo files: 递归删除文件夹中所有 nfo 文件.

### Generate thumbnails:
- 生成文件夹中视频的缩略图. 缩略图将会取视频中最有代表性的五个画面并按照 outputx.jpg 输出.

## RGB565ToHEX
- 将 RGB565 格式的图片转换为 HEX 格式并以 imagex.txt 文件输出.
